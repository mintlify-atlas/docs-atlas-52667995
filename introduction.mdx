# Welcome to Klassy

Klassy is a theming utility for the KDE Plasma desktop environment. It consists of a Global Theme which applies several components: Window Decoration, Application Style, Colour Scheme, Plasma style, and Icons. The Klassy Application Style has arguably the best scrollbars on any platform.

## What is Klassy?

Klassy is designed to be pragmatic for long-term everyday use, not for instant likes on social media or "ricing" sites. The main **Kite** theme is the result of evolving the KDE Breeze theme to be arguably more polished and usable, with influences from the original Breeze design, the "Blue Ocean" refresh, and the original KDE 1. A kite floats in the breeze!

<Note>
  Klassy requires **Plasma 6.3 or later**. Check your version in KInfoCenter before installing.
</Note>

## Why Choose Klassy?

### Pragmatic Design
The default Kite theme is designed for long-term everyday use with thoughtful design decisions that prioritize usability and polish over fleeting trends.

### Pixel-Perfect Icons
Klassy provides numerous built-in perfectly-pixel-snapped button icons, as well as the ability to display `window-*-symbolic` system icons.

![Button icon menu](https://raw.githubusercontent.com/paulmcauley/klassy/plasma6.3/images/button_icon_menu.png)

### Application Consistency
The Klassy Application Style ensures icons in dockable panels, MDI applications, and application tabs remain consistent.

## Customization Options

Klassy is highly customizable - you can make your system as classy or unclassy as you like!

The **Klassy Settings** GUI application provides easy customization. Run `klassy-settings --help` for command-line options to import Preset files, load Presets, and generate system icons.

<Tip>
  Try the bundled Presets in Klassy Settings before fine-tuning individual options.
</Tip>

![Preset selection](https://raw.githubusercontent.com/paulmcauley/klassy/plasma6.3/images/presets_opal_fruits.png)

### What You Can Customize

- **Button Icons**: Choose from perfectly-pixel-snapped built-in icons or system icons
- **Button Styles**: Integrated Rounded Rectangles, Traffic Lights, Full-height shapes, Circular highlights
- **Colors**: Translucent outlined accent colors, system color-scheme inheritance, titlebar/border matching
- **Spacing & Sizing**: Button spacing, titlebar margins, corner radius - all with HiDPI scaling
- **Transparency**: Configurable opacity for active/inactive titlebars with optional blur
- **Window Outlines**: Thin outlines with optional accent colors
- **Scrollbars**: Extensive scrollbar customization options
- **Animations**: Configurable button and window animations
