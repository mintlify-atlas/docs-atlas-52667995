# Quick Start Guide

This guide will help you get Klassy up and running with the main **Kite** global theme.

<Note>
  Make sure you have [installed Klassy](/installation) and are running Plasma 6.3 or later before proceeding.
</Note>

---

## Applying the Kite Theme

After installing Klassy, the recommended first step is to apply the main **Kite** global theme.

<Warning>
  You **must** initially use the Global Theme settings screen to apply the theme properly. If you initially apply via the Quick Settings screen or the Light/Dark selector, the full Desktop & window layout will not be applied.
</Warning>

<Steps>
  <Step title="Open System Settings">
    Launch KDE System Settings from your application menu or by pressing `Alt+Space` and typing "System Settings".
  </Step>
  
  <Step title="Navigate to Global Theme">
    Go to **Colours & Themes → Global Theme**
  </Step>
  
  <Step title="Select a Kite variant">
    Choose one of the four Kite theme variants based on your preference:
    
    - **Kite Dark - Left Panel** (recommended for 16:9 laptops)
    - **Kite Dark - Bottom Panel** (recommended for everything else)
    - **Kite Light - Left Panel**
    - **Kite Light - Bottom Panel**
  </Step>
  
  <Step title="Apply the theme">
    Click **Apply** to activate the theme. The changes will be applied immediately.
  </Step>
</Steps>

### Theme Layout Options

Klassy provides two main panel layouts:

#### Left Panel Theme
Recommended for **16:9 laptops** - maximizes vertical space for content.

## Exploring Klassy Settings

Klassy comes with a powerful configuration application called **Klassy Settings** that allows you to customize every aspect of your theme.

### Launching Klassy Settings

You can launch Klassy Settings in several ways:

<CodeGroup>

```bash Command Line
klassy-settings
```

```bash With Options
# View all available command-line options
klassy-settings --help
```

</CodeGroup>

Or search for "Klassy" in your application launcher.

### Klassy Settings Features

The Klassy Settings application provides:

- **Preset Management**: Load and import pre-configured style presets
- **Button Customization**: Icon selection, shapes, colors, and spacing
- **Window Decoration**: Titlebar styling, transparency, and borders  
- **Animations**: Configure button and window animations
- **System Icon Generation**: Create matching system icons

### Command-Line Options

Klassy Settings supports several command-line options:

```bash
klassy-settings --help          # Show all available options
klassy-settings --import <file> # Import a preset file
klassy-settings --load <preset> # Load a specific preset
```

## Trying Built-in Presets

<Tip>
  Before fine-tuning individual settings, try the bundled presets to find a style you like. Presets provide perfectly-configured combinations of settings.
</Tip>

<Steps>
  <Step title="Open Klassy Settings">
    Launch the Klassy Settings application.
  </Step>
  
  <Step title="Navigate to Presets">
    Look for the Presets section or button in the interface.
  </Step>
  
  <Step title="Browse available presets">
    Klassy includes several built-in presets:
    
    - **Kite** - The default, pragmatic theme
    - **ClassiK v3** - Full-height rounded rectangles

    - **Classik-Aurorae** - Circular button highlights
    - **Glassy Klassy** - Translucent, modern appearance
    - **Chroma** - Colorful accent-based theme
    - And many more...
  </Step>
  
  <Step title="Apply a preset">
    Click on a preset to preview it, then apply to activate.
  </Step>
</Steps>

## Enabling the Application Style

For consistent styling across all applications, make sure to enable the Klassy Application Style:

<Steps>
  <Step title="Open System Settings">
    Launch KDE System Settings.
  </Step>
  
  <Step title="Navigate to Application Style">
    Go to **Appearance → Application Style**
  </Step>
  
  <Step title="Select Klassy">
    Choose **Klassy** from the list of available application styles.
  </Step>
  
  <Step title="Apply">
    Click **Apply** to activate the Klassy application style.
  </Step>
</Steps>

This ensures icons in dockable panels, MDI applications, and application tabs are consistent.

## Configuring System Icons

For GTK applications and context menus to match your titlebar icons:

<Steps>
  <Step title="Enable Klassy icon theme">
    Go to **System Settings → Appearance → Icons** and select the **Klassy** icon theme.
  </Step>
  
  <Step title="Generate system icons">
    Open Klassy Settings and click **System Icon Generation...** to configure the icons to match your window decoration settings.
  </Step>
  
  <Step title="Apply changes">
    The generated icons will be applied to match your current Klassy configuration.
  </Step>
</Steps>

## Troubleshooting

<Warning>
  If the theme doesn't apply correctly, make sure you used **System Settings → Colours & Themes → Global Theme** for the initial application, not the Quick Settings panel.
</Warning>

### Common Issues

**Theme not applying completely**
- Ensure you're on Plasma 6.3 or later
- Apply the theme through the Global Theme settings screen, not Quick Settings
- Log out and log back in if changes don't appear

**Icons don't match**
- Enable the Klassy icon theme in System Settings → Icons
- Run System Icon Generation in Klassy Settings

**Scrollbars look wrong**
- Ensure you've enabled the Klassy Application Style
- Check that the application is using Qt/KDE frameworks

For more help, visit the [Klassy GitHub repository](https://github.com/paulmcauley/klassy) or check existing issues and discussions.
